Header:
    Shapes Count | Data Count | Material Count | Extra Count

Shapes:
    Type | Shape Data Offset | Material Data Offset | Extra Offset

Shape Data:
    [Sphere]
        X | Y | Z | R

Material:
    R | G | B | Ext

Extra:
    Sphere:
        Mat Type | Extra Count | |

    Cube:
        Mat Type | Extra Count | |
        sX | sY | sZ |
        rX | rY | rZ
